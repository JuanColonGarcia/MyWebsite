My Portfolio Some Of My Works

Credit Card Detection

This program implements a neural network model to classify credit card transactions as legitimate or fraudulent using a simulated dataset. The dataset contains transactions from January 1, 2019, to December 31, 2020, and includes both legitimate and fraud transactions from 1,000 customers interacting with a pool of 800 merchants.

Plane price estimation

The objective of this project is to predict flight prices using a dataset obtained from the "Ease My Trip" website. To achieve this, we will employ the 'Linear Regression' and random forest algorithms, which are effective for training the dataset and predicting continuous target variables. "Ease My Trip" is an online platform where potential passengers can book flight tickets, making it a valuable resource for travelers.

Voice Recognition

An app that recognizes your voice and changes the background color and a figure based on what you tell the computer.

Metaprogram

A metaprogram capable of generating the control flow graph for different programs using the JavaParser library.

AndroidProject

This Android application allows users to interact with a dynamic environment where images of different objects (such as trees, benches, rivers, and rocks) can be added, moved, and resized on the screen.

chatbot

Conversational bot that recommends movies. It uses the services of a CLUIS App published as an Azure resource.

RESTful Services with Spring Boot

This project involves creating RESTful services with Spring Boot and developing Python/Java clients to consume them. It covers REST service implementation, task list management (ToDoList), database access with multi-layer architecture, and query parameters in RESTful APIs.
